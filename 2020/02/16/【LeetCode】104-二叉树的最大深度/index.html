    二叉树的最大深度

    题目

    给定一个二叉树，找出其最大深度。
    二叉树的深度为根节点到最远叶子节点的最长路径上的节点数。

    说明: 叶子节点是指没有子节点的节点。
    示例：
    给定二叉树 [3,9,20,null,null,15,7]（示例图已省略，可点击顶部标题查看原题），返回它的最大深度 3 。

    解法1：迭代法

    • 解题思路
      迭代法的实现和层序遍历相似，每遍历完一层，深度+1，当遍历完成时便可得到最大深度。
      和层序遍历不同的是，需要记录下一层的节点个数 levelSize，当 levelSize 为 0 时，则意味着已遍历完一层，深度+1。
